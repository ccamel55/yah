# Yet another hook 

Cross-platform, cross architecture function hooking library.

## Hooking methods

| Method             | Implemented |
|--------------------|-------------|
| VFunc replacement  | No          | 
| VTable replacement | No          |
| Inline/trampoline  | No          |

## Platforms

| OS      | x86 | x86_64 | Arm64/aarch64 |
|---------|-----|--------|---------------|
| Linux   | No  | No     | No            |
| Windows | No  | No     | No            |
| MacOS   | No  | No     | No            |
